Placeholder Text

One of the most common ways of labeling form inputs is using Placeholder Text. A placeholder label accurately describes the purpose of the form input, but is not visible after a user enters text.

Input & Label

Using a label element in conjunction with an input solves this problem. When text is entered in the input, the label remains visible. This can be especially helpful for users filling out long forms.

The best of both worlds?

A great solution is using floating labels. On the surface, this looks much like our first example, but when a user enters text, the label animates slightly above it's input.
